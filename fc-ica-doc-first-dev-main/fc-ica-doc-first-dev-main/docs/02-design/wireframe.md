## 1. 로그인 화면

### 레이아웃

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│                                                             │
│                    ┌───────────────────┐                    │
│                    │                   │                    │
│                    │    TaskFlow ◆     │                    │
│                    │                   │                    │
│                    │  ┌─────────────┐  │                    │
│                    │  │ 이메일       │  │                    │
│                    │  └─────────────┘  │                    │
│                    │                   │                    │
│                    │  ┌─────────────┐  │                    │
│                    │  │ 비밀번호     │  │                    │
│                    │  └─────────────┘  │                    │
│                    │                   │                    │
│                    │  ┌─────────────┐  │                    │
│                    │  │   로그인     │  │                    │
│                    │  └─────────────┘  │                    │
│                    │                   │                    │
│                    │  계정이 없으신가요? │                    │
│                    │  회원가입 →       │                    │
│                    │                   │                    │
│                    └───────────────────┘                    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 컴포넌트 상세

| 컴포넌트 | 타입 | 설명 |
|----------|------|------|
| 로고 | Text + Icon | "TaskFlow" 텍스트 로고, 중앙 정렬 |
| 이메일 입력 | Input | type="email", placeholder="이메일 주소" |
| 비밀번호 입력 | Input | type="password", placeholder="비밀번호" |
| 로그인 버튼 | Button (Primary) | 전체 너비, 비활성화 상태(입력 전) |
| 회원가입 링크 | Text Link | 하단 중앙 정렬 |

### 인터랙션

1. 이메일/비밀번호 모두 입력 시 로그인 버튼 활성화
2. 로그인 실패 시 입력 필드 상단에 에러 메시지 표시 (빨간색)
3. 로그인 중 버튼에 로딩 스피너 표시
4. Enter 키로 로그인 가능

### 화면 전환

- **로그인 성공** → 대시보드 (`/dashboard`)
- **회원가입 링크 클릭** → 회원가입 화면 (`/register`)

- **새 프로젝트 버튼** → 프로젝트 생성 모달 (현재 화면 위에 오버레이)

### 컴포넌트 상세

| 컴포넌트 | 타입 | 설명 |
|----------|------|------|
| 보드 헤더 | SubHeader | 프로젝트명, 뒤로가기, 설정, 멤버 버튼 |
| 필터 바 | FilterBar | 담당자, 우선순위 드롭다운 필터 |
| 컬럼 | Column | 상태명, 태스크 수, 태스크 카드 목록 |
| 태스크 카드 | TaskCard | 우선순위 뱃지, 제목, 담당자 아바타 |
| 추가 버튼 | Button (Ghost) | 컬럼 하단, 클릭 시 인라인 입력 또는 태스크 생성 모달 |

### 우선순위 뱃지 색상

| 우선순위 | 색상 | 아이콘 |
|----------|------|--------|
| URGENT | 빨강 (🔴) | 위쪽 화살표 2개 |
| HIGH | 주황 (🟠) | 위쪽 화살표 1개 |
| MEDIUM | 노랑 (🟡) | 가로줄 |
| LOW | 초록 (🟢) | 아래쪽 화살표 |

### 인터랙션

1. 태스크 카드 드래그 시작 → 카드 반투명 + 드롭 영역 하이라이트
2. 컬럼에 드롭 → 상태 변경 API 호출, 낙관적 업데이트
3. 태스크 카드 클릭 → 사이드 패널 열림 (오른쪽에서 슬라이드)
4. 컬럼 하단 "+" 클릭 → 태스크 빠른 생성 (인라인 제목 입력)

### 화면 전환

- **← 대시보드 클릭** → 대시보드 (`/dashboard`)
- **태스크 카드 클릭** → 태스크 상세 사이드 패널 (현재 화면 위에 오버레이)
- **설정 클릭** → 프로젝트 설정 모달
- **멤버 클릭** → 멤버 관리 모달

### 컴포넌트 상세

| 컴포넌트 | 타입 | 설명 |
|----------|------|------|
| 닫기 버튼 | IconButton | ✕ 아이콘, 패널 닫기 |

| 삭제 버튼 | Button (Danger) | 확인 다이얼로그 후 삭제 |
| 저장 버튼 | Button (Primary) | 변경사항 저장 |

### 인터랙션

1. 패널 외부 클릭 또는 ESC 키로 닫기
2. 상태 변경 시 보드뷰의 카드 위치가 실시간 업데이트
3. 삭제 클릭 시 확인 다이얼로그 표시
4. 저장 성공 시 토스트 메시지 "저장되었습니다" 표시
